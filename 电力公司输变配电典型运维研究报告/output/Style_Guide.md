# 《电力公司输变配电典型运维研究报告》风格指南

*   **语调规范**
    *   保持绝对客观与技术严谨性，剔除所有情绪化表达与营销辞令（如“卓越”、“颠覆”）。
    *   文风需干练、紧凑，优先使用陈述句，确保决策层能快速获取高密度信息。

*   **结构逻辑**
    *   遵循“数据现状 -> 归因分析 -> 解决方案”的逻辑闭环。
    *   **数据先行**：每章节必须以关键 KPI（如故障停运率、运维成本率）开篇。
    *   **结论清晰**：段落结尾需提供可执行的具体技术或管理建议，拒绝模棱两可。

*   **数据约束**
    *   数据必须真实可溯源，严格标注出处（如：电网年报、CIGRE、IEEE 标准），严禁模糊估算或虚构。
    *   全文严格限制为 **1 张关键参数对比表** 与 **1 张核心趋势图**，拒绝无效数据堆砌。

*   **语言准则**
    *   实施“零废话”原则，删除修饰性形容词，多用动词与名词。
    *   精准使用行业术语（如：状态检修 CBM、N-1 准则、馈线自动化），避免同一概念的重复阐述。

*   **图表要求**
    *   仅保留 **一张** 最具分析价值的核心图表（如：全生命周期成本分析图）。
    *   图注需精炼至 20 字以内，直接揭示数据背后的核心洞察，而非单纯描述现象。
